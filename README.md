# Automatic Image Slider

A simple automatic image slider built using HTML and CSS.  
This project was created to practice basic animation techniques and slideshow layouts for front-end web interfaces.

## Technologies Used
- HTML5
- CSS3

## Features
- Automatic sliding images
- Smooth transition animation
- Clean and simple layout
- Lightweight front-end implementation

## Purpose of the Project
This project was built as part of my front-end development practice to improve my understanding of CSS animations and layout techniques for image sliders.
